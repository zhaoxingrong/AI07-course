Convolutional Neural Networks

CNN：主要应用在计算机视觉领域。通过一些 filter，不断地提取特征，从局部特征到总体特征，达到图像识别等功能。每一个图像数学上都是矩阵，每一个视频的连续的图像。

**channel 通道**：主要是三色素 RGB。

filter: 一组数字，也就是权重参数。

padding 填白：每次卷积前，先给图片周围补一圈空白，让卷积之后图片跟原来一样大。

  • 每次卷积 (N-F+1 维) 导致图像缩小，信息丢失。
  • 图片中间信息相对于边缘信息过度使用。

stride 步长：filter 在原图上扫描时，需要跳跃的格数。

  • stride 过小，重复计算较多，计算量大，训练效率降低。
  • stride 过大，信息遗漏，无法有效提炼数据背后的特征。

convolution 卷积：

M=floor[N−F+2PS]+1M = floor [ \frac{N- F + 2P}{S} ]+ 1M=floor[SN−F+2P​]+1

其中 NNN 是输入的数据大小，FFF 是 filter，PPP 是 padding, SSS 是 stride。

pooling 池化：为了提取一定区域的主要特征，并减少参数数量，防止模型过拟合。

  • Max Pooling
  • Average Pooling

CNN 优点：

  • 参数共享，减少参数个数，避免过拟合。
  • 连接的稀疏性，减少不同区域对特征的影响。

常见 CNN:

  • LeNet
  • AlexNet
  • VGG
  • GoogleNet
  • ResNet
